# Lista de Clientes com POO 📋

Projeto desenvolvido como atividade acadêmica com o objetivo de aplicar os conceitos de Programação Orientada a Objetos (POO) em JavaScript por meio de um sistema simples de cadastro e listagem de clientes.

## 📋 Sobre o Projeto

A aplicação permite cadastrar clientes e exibir os dados na interface de forma dinâmica.

O foco principal do projeto foi a utilização de conceitos de Programação Orientada a Objetos para organizar melhor o código, tornando-o mais estruturado, reutilizável e fácil de manter.

## 🚀 Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* Programação Orientada a Objetos (POO)

## ✨ Funcionalidades

* Cadastro de clientes
* Exibição dinâmica da lista de clientes
* Organização dos dados em objetos
* Atualização automática da interface
* Estrutura de código orientada a objetos

## ▶️ Como Executar

1. Clone o repositório:

```bash
git clone https://github.com/brunakm/lista-clientes-poo-.git
```

2. Abra o arquivo `index.html` em qualquer navegador.
